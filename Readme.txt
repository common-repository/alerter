Add an alert or notification in your WordPress website, without having to code anything. Fast and easy.

== Description ==
The Alerter plugin is the easiest way adding a notification or alert to your Website. Just using a shortcode [alerter].
The plugin allows you to easily add text or notice, change the text color, border color and background color through a Colorpicker,
the shortcode [alerter] can be added to a post, page, or widget. This plugin is a beta version, and will be actively developed.

[youtube https://youtu.be/PRXS7DUvUbw]

== The idea ==
When building custom websites for clients, one of them asked for a simple solution, publishing a notice on his website. The no-technical client can adding a notice without coding.

On our roadmap:

* Change border radius
* Add popup option (Pro)

== Installation ==
1. Unpack and Upload the alerter.zip plugin in the to the "/wp-content/plugins/" directory, or upload the plugin zip via the plugins settings page.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Add your alert text in the Alerter-settings page and choose your colors.
4. Add the shortcode [alerter] where you want on your WordPress website
